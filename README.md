# Retail SQL Data Modelling & KPI Calculations

This project focuses on building a relational SQL data model for a retail business scenario and writing analytical SQL queries to calculate business KPIs.

It simulates a real-world retail dataset including customers, products, orders, and order items.

# Key Features

* Database schema design
* Primary/foreign key relationships
* Data normalisation
* SQL joins & aggregations
* KPI calculations
* Window functions
* CTE-based analytical queries

# Technologies Used

* SQL
* PostgreSQL / MySQL / SQL Server (any supported engine)

# Project Structure

/data         → sample datasets  
/sql          → SQL scripts (schema, KPIs, queries)  
/visuals      → ERD diagrams or outputs
